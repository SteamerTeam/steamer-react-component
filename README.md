# steamer-react-component

用于开发 `react` 组件的脚手架


## 如何使用

* 直接从github clone 下来
* 使用[steamerjs](https://github.com/SteamerTeam/steamerjs)安装

```javascript
// 安装依赖
npm i

// 开发
npm run start 或 npm run dev
// 打开链接，查看 demo
localhost:9000

// 查看开发环境源码
npm run source

// 代码规范扫描
npm run lint

// 测试
// 使用 jest 测试
npm run test 或 npm run jest
// 使用 karma 测试
npm run karma

// 生产代码生成
// 使用 babel 编译
npm run dist 或 npm run babel
// 使用 webapck 编译
npm run webpack

```
